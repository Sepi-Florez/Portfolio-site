Half-a-minute

This is an app I made as a tool to play the board game 30 Seconds. This app allows me to use a custom made database of words to play the game with.  

The database of words can easily be changed through excel, Each horizontal collumn in Excel counts as a category of which the first word is the category name. The app wil then read a CSV. version of the Excel file and it will then show up in the list of categories inside the app.
